# API Reference

Complete reference documentation for all streamz processors and types.

## Core Types

### Result[T]

The fundamental type for stream processing, encapsulating success values or errors.

```go
type Result[T any] interface {
    IsSuccess() bool
    IsError() bool
    Value() T
    Error() *StreamError[T]
}
```

### StreamError[T]

Error type with context preservation.

```go
type StreamError[T any] struct {
    Item          T
    Err           error
    ProcessorName string
}
```

## Common Patterns

All processors follow this signature pattern:

```go
Process(ctx context.Context, in <-chan Result[T]) <-chan Result[Out]
```

Time-based processors support clock injection for deterministic testing:

```go
processor.WithClock(clock clockz.Clock)
```
